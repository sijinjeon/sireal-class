# 방법·순서

1. 후보 **나열**
2. 후보별 **딥리서치** → 근거 수집
3. **심사 기준** 적용 → 점수
4. **우선순위** → **1개 확정**

> 딥리서치는 시간이 걸린다 → 후보 정해지는 대로 **바로 돌려두기**

**심사 기준 5가지** — 문제 정의·창의성·AI 활용·실현 가능성·기대 효과

# 규칙

- **심사위원** 관점·기준으로 본다
- "이거 좋은데요?" 반응 = **신호**
- 딥리서치는 후보 확정 즉시 실행
- 결과 + 점수를 **노션에 함께** 기록
- **선정 이유**를 문서에 남긴다
