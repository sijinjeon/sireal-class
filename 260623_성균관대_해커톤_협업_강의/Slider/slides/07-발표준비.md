# 7. 발표준비 — 슬라이드 3단계

1. **문서 정리·목차 구성**
   - PRD, Changelog, README → 핵심 메시지·목차 확정
2. **세부 내용 작성**
   - 확정 목차별 문장·수치·시연 포인트
3. **디자인 적용**
   - 내용 + **디자인 가이드** → 슬라이드 완성

> 목차·내용 **먼저**, 디자인은 **마지막**

---

<!-- _class: body compact -->

# 7. docs/ → 발표 · 도구

| 문서 | 발표에서 |
|------|---------|
| PRD | 문제·기능·기대효과 |
| Changelog | "우리가 만든 것" |
| README | 프로젝트 소개 |
| 디자인 가이드 | 슬라이드 톤·색 |

**도구** — Gemini, NotebookLM, Claude, Canva, Gamma 등

---

<!-- _class: body -->

# 7. 발표준비 — 규칙 · 실습

- `docs/`에서 **시작** (새로 쓰지 않는다)
- **목차 먼저** 확정
- **3단계 순서** 지키기
- AI 결과 ↔ **docs/ 대조**
- **시연 리허설** + **시간 맞추기**

### 실습
PRD·Changelog @첨부 → 목차 5~7장 → 내용 → 디자인 1장
