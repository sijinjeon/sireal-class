05 — Animation
진입 애니메이션

슬라이드가 활성화되면 .a 클래스가 붙은 요소들이 순서대로 나타납니다.
HTML 요소에 class="a"만 추가하면 자동으로 애니메이션됩니다.

/* 진입 애니메이션: 요소에 class="a" 추가 */ <h2 class="a lg">제목</h2> <p class="a">본문</p> <table class="a data-table">...</table> /* .a 는 슬라이드 활성화 시 JS가 순차적으로 visible 처리 */ /* 순서 = DOM 순서 그대로 (위→아래) */
06 — 새 슬라이드 추가하는 법

1.  index.html 에서 원하는 위치에 아래 블록을 복사해 붙여넣습니다.
2.  섹션명 · 제목 · 내용을 바꿉니다.
3.  진입 효과가 필요한 요소에 class="a"를 추가합니다.
4.  슬라이드 총 개수는 JS가 자동으로 계산합니다.
